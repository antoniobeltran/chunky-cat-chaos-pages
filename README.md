# Chunky Cat Chaos — App Store Pages

Static pages for App Store Connect:

- Privacy Policy
- Support

## Publish on GitHub Pages

1. Create a new public repo on GitHub named `chunky-cat-chaos-pages`.
2. Push this folder:

```bash
git remote add origin https://github.com/antoniobeltran/chunky-cat-chaos-pages.git
git push -u origin main
```

3. On GitHub: **Settings → Pages → Build and deployment**
4. Source: **Deploy from a branch**
5. Branch: **main** / **/ (root)** → Save

## App Store Connect URLs

After Pages is live:

- Privacy Policy: `https://antoniobeltran.github.io/chunky-cat-chaos-pages/privacy.html`
- Support: `https://antoniobeltran.github.io/chunky-cat-chaos-pages/support.html`
